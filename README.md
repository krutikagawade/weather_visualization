Project: API Integration and Data Visualization

Description:
This project fetches weather data from the OpenWeatherMap API using Python and visualizes temperature and humidity using Matplotlib and Seaborn.

Technologies Used:
Python
Requests
Matplotlib
Seaborn
Pandas

Steps:
1. Get API key from OpenWeatherMap
2. Run the Python script
3. Weather data is fetched
4. Visualization graph is displayed
